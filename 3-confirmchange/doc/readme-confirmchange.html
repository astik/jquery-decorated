confirmchange

Basics

Some user like to have a confirmation message before submitting any information with a form. It is a common thing with delete form : you have a table displaying one entities on each row and a checkbox is available to select the one you want to delete.

This plugin is really simple, it juste display a confirm dialog when the form is submitted.

Demo

You can see this in action

Options

  • selector : (default ":input") selector of current form's input to protect from changes
  • confirmMsg : (default "Are you sure you want to submit the modification ?") message to display in the confirm box
  • displayDialog : [function] function that display the dialog box. if you don't like the basic javascript confirm dialog, you can override this function to display a much nicer dialog box than the confirm one.

Exemple

Here is the code to use confirmchange plugin on each input of the selected form :

$("form").confirmchange();

Here is the code to use confirmchange plugin on each text input of the selected form :

$("form").confirmchange({ selector: "input[@type=text]" });
